# Meal Generator

I often struggle to decide on what to eat on any given night and enjoy finding new recipes to cook so I started this project for the purpose of generating dinner ideas. It is a simple app that fetches and returns information from an API such as an ingredient list, instructions on how to make the dish, and a youtube video tutorial for making the dish. Bon app√©tit.
